[02] CORE_FUNCTIONALITY_V1.5
  • 1. Latency & Input: Timer resolution via ntdll.dll and real-time Mouse Hz via Raw Input to ensure rated peripheral speeds.
  • 2. Performance Monitoring: Detects Power Throttling, monitors Core Parking, Processor Queue, and Context Switches to identify background noise.
  • 3. Advanced Resource Audit (PERF): Monitors Disk Queue, Response Time (ms), Page Faults, RAM, Standby List, and UDP receive errors.
  • 4. Kernel Tracing (ETW): DPC/ISR audit via Event Tracing for Windows with auto-logging of spikes exceeding 500µs.
  • 5. System Diagnostics: Quick hardware view (OS, CPU threads, Disks) and automatic full system scan upon launch.
