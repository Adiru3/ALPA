[02] CORE_FUNCTIONALITY
  • Timer Resolution: High-precision monitoring via ntdll.dll.
  • Mouse Polling Rate: Real-time Hz calculation using Raw Input (WM_INPUT).
  • Processor Queue: Detects if CPU threads are bottlenecked by too many processes.
  • Context Switches: Monitors frequency of thread switching to identify background "noise".
  • Kernel Tracing (ETW): DPC/ISR audit for problematic drivers using Event Tracing for Windows.
  • Spike Detection: Auto-logs any DPC spike exceeding 500µs in red.
  • System Diagnostics: Quick view of OS version, CPU threads, and disk models (NVMe/SSD/HDD).
  • Auto-Audit: Application initiates a full system scan immediately upon launch.
  • Таймер системы: Высокоточный мониторинг разрешения через ntdll.dll.
  • Частота мыши: Расчет Гц в реальном времени через Raw Input (WM_INPUT).

  • Переключения контекста: Мониторинг частоты смены потоков для поиска фонового «шума».
